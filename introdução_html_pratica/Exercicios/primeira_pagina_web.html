  1. Qual a importancia das tags

    2. O conhecimento sobre tags é de suma importancia para o programador front-end, visto que elas são a base para qualquer estrutura web. Além disso, elas dão significado semântico a estrutura do HTML. Com isso em mãos, com o estudo e aplicação delas, o programador(a) consegue não só entende-las, mas tambem consegue aumentar sua produtividade, visto que com a repetição e a prática, ele já sabe como usá-las e como aplicar em cada cenário em que se encontra.

    Além disso, muitos problemas comuns do front-end podem ser resolvidos com uma melhor compreensão das tags HTML. Por exemplo, saber como usar corretamente as tags de formulário facilita a criação de formulários interativos. Da mesma forma, o conhecimento das tags de mídia simplifica a incorporação de áudio e vídeo.

  2. Tags principais

    3. As principais tags são:

    • h1-h6:

      • As tags h1 até h6 são usadas para definir os cabeçalhos de um documento HTML. h1 é usado para o cabeçalho principal, enquanto h2 até h6 são usados para subcabeçalhos. Usar essas tags corretamente estruturam o conteúdo de uma página, melhorando a acessibilidade e ajudando os mecanismos de busca a entender a hierarquia do conteúdo.

    • p

      • A tag p é usada para definir um parágrafo. É um bloco de conteúdo que quebra automaticamente as linhas de acordo com o espaço disponível. É uma parte essencial da estruturação do texto em um documento HTML.

    • a

      • A tag é usada para criar hiperlinks, permitindo que os usuários cliquem e sejam direcionados para outra página ou seção do site. Esta tag é crucial para a navegação e a interatividade de um site.

    • ul e ol

      • As tags ul e ol são usadas para criar listas em HTML. ul é usado para listas não ordenadas (bullets), enquanto ol é usado para listas ordenadas (numeradas). Ambos são usados com a tag li para definir cada item da lista.

    • hr

      • A tag ht cria uma linha horizontal na página para criar, por exemplo, divisões entre os temas apresentados, como foi usada nessa pagina no inicio. Você pode ver ela proximo ao tema Desbravando as tags no html"

    • del

      • A tag del é a tag que permite representar a exclusão de um texto ou termo. Ele passa uma linha no meio da palavra ou texto escolhido e assim, representa a exclusão do termo.

    • small

      • A tag small deixa os textos pequenos, como comentários do autor no final de uma página.

    • i

      • A tag i faz os textos ficarem no formato itálico.

    • li

      • A tag li permite lista os itens de uma lista, seja ela organizada ou não. É muito usada com as tags ul e ol , ou seja, listas desordenadas e ordenadas, respectivamente.

    • strong

      • A tag strong nos permite deixar as palavras em negrito.

    • sub e sup

      • A tag sub e sup especifica texto embutido que deve ser exibido como subscrito e subrescrito, respectivamente, por razões exclusivamente tipográficas. Os subscritos são normalmente renderizados com uma linha de base reduzida usando texto menor. Já os sobrescritos geralmente são renderizados com uma linha de base elevada usando texto menor.

    • blockquote
      • Essa tag indica que o texto incluído é uma longa citação.
